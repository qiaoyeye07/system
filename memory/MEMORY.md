## v1.3 更新摘要（2026-07-08）

本次更新合并了"细节规则补充"和"举报功能"两个分支，主要变更：

### 合并冲突解决（4 份基线文档）
- 合并版本描述：涵盖退款中状态、交换订单异常流程、自动完成时间、举报功能
- 管理员导航：移除"发布商品"入口 + 新增"举报处理"入口
- 普通用户导航：新增"我的举报"入口

### 举报功能（新增）
- **PRD**：新增 §6.10 举报功能（FR-RPT-01 至 08）
- **SRS**：新增 SRS-RPT 需求编号、BR-16/17/18 业务规则
- **页面说明**：新增 UI-USER-11（我的举报页）、UI-ADM-04（举报处理页）
- **原型**：新增 §5.6 举报表单、§5.7 我的举报页、§5.8 举报处理页线框图
- **数据模型**：新增 `report` 表（v1.1）
- **接口设计**：新增 9 个举报 API + 4 个管理端举报 API（v1.1，总计 56 接口）
- **测试方案**：新增 11 个举报测试用例（v1.1）
- **实施计划**：阶段 6 扩展为"评价、纠纷与举报"（+5 个任务，+3 天，+5 人天）
